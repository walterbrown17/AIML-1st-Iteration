Unordered and Ordered Lists - Best Practices

Unordered List - Key Points

  • Always use <ul> for lists where the order doesn't matter.
  • Each list item should be wrapped in an <li> tag.
  • Indent child list items properly for readability.
  • Use semantic lists to improve accessibility and SEO.

Ordered List - Key Points

  1. Use <ol> when the sequence of items is important (e.g., steps).
  2. Ensure that each step is clear and concise inside an <li> tag.
  3. Maintain proper nesting if sublists are needed within an item.
  4. Avoid excessive list nesting to keep the HTML structure clean.

CSS Syntax and Selectors

Syntax of CSS:

selector {
    property: value;
}

How to apply CSS:

  • Inline CSS: Written directly in the HTML tag using style attribute as style="property:value;", you can find the example in the body tag
  • Internal CSS: Written inside a <style> tag in the <head>
  • External CSS: Written in a separate .css file and linked to the HTML

Types of CSS Selectors

  • Tag Selector: Targets all elements of a given tag (p {})
  • Class Selector: Targets elements with a class attribute (.note {})
  • ID Selector: Targets a single unique element (#mainTitle {})
  • Parent > Child Selector: Targets direct children (section > p)
  • Ancestor Descendant Selector: Targets nested elements (section p)
  • Combined Selector: Targets elements with multiple conditions (p.note, #mainTitle)

Understanding Cascading, Specificity, and the Box Model

Cascading in CSS

The term Cascading means multiple CSS rules can apply to the same element, and the browser must decide which one wins.

  • Source Order – Last rule wins if specificity is the same.
  • Specificity – ID > Class > Tag selector.
  • Important – !important overrules everything (avoid overuse).

Understanding <div> and CSS Flexbox

<div> Element

The <div> tag is a block-level element used to group content. It is primarily used for layout and styling with CSS.

Flexbox Layout

Flexbox is a CSS layout model that distributes space and aligns items efficiently. It's ideal for building responsive UIs where items adjust dynamically.

  • display: flex; → turns a container into a flexbox.
  • justify-content: controls horizontal spacing.
  • align-items: controls vertical alignment.
  • flex-wrap: allows items to wrap on small screens.
